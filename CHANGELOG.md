## [1.0.0]

### Added
- Syntax highlighting for `.scenario` and `.fragment` files
- Code completion for OpenAPI operations, fragments, and keywords
- Go to Definition for fragments and OpenAPI operations
- Find References for fragments and steps
- Document symbols (outline view) for scenario structure
- Hover information for operations and keywords
- Code folding for scenarios and steps
- Document formatting
- Clickable links in documents
- OpenAPI specification integration
- Code snippets for common patterns

### Technical
- Requires VS Code 1.118.0+
- Uses TypeScript with ESLint
- TextMate grammar for syntax highlighting
