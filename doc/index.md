# BerryCrush VS Code Extension

Welcome to the BerryCrush VS Code extension documentation. This extension provides IDE support for BerryCrush scenario and fragment files.

## Features

- **[Syntax Highlighting](features/syntax.md)** - Rich syntax highlighting for `.scenario` and `.fragment` files
- **[Navigation](features/navigation.md)** - Go-to-definition and find references for fragments and operations
- **[Auto-Completion](features/completion.md)** - Context-aware completions for keywords, operations, and fragments
- **[Code Snippets](features/snippets.md)** - Quick snippets for common patterns
- **[Formatting](features/formatting.md)** - Document formatting with table alignment
- **[Code Folding](features/folding.md)** - Collapse scenarios, fragments, and step blocks
- **[Document Outline](features/outline.md)** - Symbols view for easy navigation
- **[Document Links](features/links.md)** - Clickable links to fragments and operations
